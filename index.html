Image Text Extractor

How does it work?

You select an URL to a image with text. A request is send to Microsoft Cognitive API. The request is send back here.

Image Restrictions

  • Supported image formats: JPEG, PNG, GIF, BMP.
  • Image file size must be less than 4 MB.
  • Image dimensions must be at least 50 x 50.
